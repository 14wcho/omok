# 오목 게임

브라우저에서 바로 실행되는 오목 게임입니다.

🎮 **플레이**: https://14wcho.github.io/omok

---

## 기능

- **온라인 대전** — 방 코드로 멀리 있는 상대와 실시간 대전 (PeerJS P2P)
- **로컬 대전** — 같은 컴퓨터에서 2인 대전
- **AI 대전** — 혼자서 AI와 대전

## 기술 스택

- HTML / CSS / JavaScript (순수 바닐라, 프레임워크 없음)
- [PeerJS](https://peerjs.com/) — 서버 없이 브라우저 간 P2P 연결
- GitHub Pages — 무료 호스팅

## 온라인 대전 방법

1. 한 명이 **"방 만들기"** 클릭 → 5자리 코드 생성
2. 상대방에게 코드 전송
3. 상대방이 코드 입력 후 **"참가하기"** 클릭
4. 연결되면 게임 시작!

## 로컬 실행

서버 없이 `index.html` 파일을 브라우저에서 바로 열면 됩니다.

```
omok/
└── index.html   # 게임 전체 (단일 파일)
```
